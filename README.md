# Hospitality Revenue Analysis

This project analyzes hotel performance using Power BI.

## KPIs Used:
- Revenue
- ADR
- RevPAR
- Occupancy %
- Realisation %

## Key Insights:
- Weekend occupancy is 12% higher than weekdays.
- Luxury segment contributes 65% of total revenue.
- Platform Logtrip has lowest realisation %, indicating high cancellations.
- RevPAR declined in Week 26 due to drop in occupancy.
